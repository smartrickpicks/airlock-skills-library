# airlock-skills-library

Agent skills, component library, and UI templates for [Airlock](https://github.com/smartrickpicks/airlock-app).

## Contents

- `skills/` — Agent skill definitions (SKILL.md format)
- `components/` — Official UI component library
- `templates/` — Page layouts and UI templates
- `moodboard/` — Design references
- `toolkits/` — CSS, JS, React utilities
- `intake/` — Submission pipeline for new content

## Pack Format

Structured as an Airlock pack. See [pack-schema.json](https://github.com/smartrickpicks/airlock-config/blob/main/pack-schema.json) for the schema.

## Architecture

Part of the Airlock constellation. See [airlock-config](https://github.com/smartrickpicks/airlock-config) for the MCP registry.
