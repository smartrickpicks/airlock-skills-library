# OTTO Skill Catalog — Top Picks per Persona

This file contains the top 10 recommended skills for each persona, along with the rationale for why they're a good match. The full catalog of 393 skills is in the HTML alpha demo.

When recommending skills, prioritize this list for the active persona. For skills not installed locally, include the install guidance.

## Skill Install Guidance

- **Airlock skills** (source: local): Available in the Airlock Skills Library
- **MCP servers** (source: external): Search the MCP marketplace or install via `claude mcp add`
- **Custom skills**: User-ingested via OTTO's Skill Ingestion feature in the HTML alpha
